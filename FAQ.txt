VSC
************
Alt + Z  форматирование длинной строки на короткие (в окне редактора)
Alt + курсор (множественное выделение)
Chift + Alt + стрелка (вниз/вверх)  - дублирование строки
Ctrl + [   сдвиг кода влево   (Chift + Tab)
Ctrl + ]   сдвиг кода вправо   (Tab)

Alt + X копирование всех классов из HTM в CSS (при установленном плагине VS HTML to CSS)
